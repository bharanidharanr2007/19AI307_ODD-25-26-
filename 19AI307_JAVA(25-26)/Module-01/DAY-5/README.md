## QUESTION:
Write a Java program to find the absolute value of a number using Math.abs().

## AIM:
To write a Java program to find the absolute value of a number using Math.abs().

## ALGORITHM :
1.	Start the program.
2.	Import the necessary package 'java.util'
3.	Create a Scanner object to read input.
4. Prompt the user to enter a number.
5. Read the input number.
6. Use Math.abs() function to find the absolute value.
7. Display the result.
8. Stop the program.

## SOURCE CODE:

```
import java.util.*;
public class Vijay{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        float value =sc.nextFloat();
        float a= Math.abs(value);
        System.out.print("Absolute value = "+a);
    }
}
```
